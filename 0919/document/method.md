# method

- method 
	- 데이터를 가공하여 쓸모 있는 정보를 만드는 행위
	- 코드의 재사용이 용이

#### static 메서드 선언과 호출

|선언과 호출|방법|
|---|---|
|메서드선언|메서드 시그네이처 사용(리턴 타입과 메서드 이름은 반드시 필요)|
|메서드호출|다른클래스에서 호출  - '클래스명.메서드명'|
||같은 클래스에서 호출 - '메서드명'|	

#### immutable vs. mutable

|종류|타입|
|---|---|
|immutable|기본, String|
|mutable|참조|
